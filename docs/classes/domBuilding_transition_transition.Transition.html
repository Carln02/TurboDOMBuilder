Class Transition

Transition

Description

A class representing a CSS transition. It has two states (in and out), which you can set up almost independently (they must only share the animation properties). Use a Transition to transition one or more HTMLElement(s) easily.

apply

  • apply(direction?, elements?, executeForAll?, recomputeIndices?, overrideStyles?, overrideBeforeComputing?): this

  • Parameters

    • Optional direction: InOut = InOut.out

      The direction of the transition.

    • Optional elements: HTMLElement | HTMLCollection | HTMLElement[]

      One or more HTMLElements to which the transition will be applied. Defaults to the transition's attached elements.

    • Optional executeForAll: boolean = false

      If set to true, the function will be applied also to all the previously attached/transitioned elements, and interpolation values will be computed from the stored list of attached elements (containing both previous and new elements).

    • Optional recomputeIndices: boolean = true

      Define whether to keep previously computed indices or recompute them based on the passed elements list.

    • Optional overrideStyles: TransitionStyles

      Optional styles to override the defaults. Set to null to not set any styles on the element(s).

    • Optional overrideBeforeComputing: TransitionInterpolation<void> = ...

      The callback to execute on all concerned elements before returning the elements' list.

    Returns this
